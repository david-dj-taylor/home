Modelling Intermittency

The New Method for Modelling Intermittency

Many water systems outside of the United States experience significant intermittency. This web interface edits INP files to account for this intermittency. Simply save you INP file as a .txt file (using notepad for Windows or textEdit for Mac) and upload it below. Then press create file and follow the download link for your new file, which can be modeled by EPANET. If you would like to customize certain variables, type in new values below and press create variable. Please note that this technology is still in beta.
